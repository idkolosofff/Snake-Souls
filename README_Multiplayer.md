#Snake Souls

##Запуск:
Зайти в корень и написать 'python main.py'
Если вдруг не запустится, открыть в VSCode папку Snake-souls и запусить main который в корне

##Функционал:
Пока минимальнейший, можно создать свой сервер, и другому человеку подключиться к нему по ip, введя его в терминале, многий функционал Single-Player змейки все еще восстанавливается, покинуть сервер можно на Escape.
